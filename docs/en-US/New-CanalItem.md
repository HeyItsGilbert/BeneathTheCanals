# New-CanalItem

## SYNOPSIS
Generate a pentolan item for your players to find.

## SYNTAX

```
New-CanalItem [<CommonParameters>]
```

## DESCRIPTION
Quickly get a list of features for an item.

## EXAMPLES

### Example 1
```powershell
C:\> New-CanalItem
```

Returns a list of features for you to create an item.
