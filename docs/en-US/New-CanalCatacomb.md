# New-CanalCatacomb

## SYNOPSIS
Create a new catacomb for your players to explore.

## SYNTAX

```
New-CanalCatacomb [[-Category] <CatacombCategory>] [[-Room] <CatacombRoom>] [<CommonParameters>]
```

## DESCRIPTION
Create a new catacomb for your players to explore.

## EXAMPLES

### Example 1
```powershell
> New-CanalCatacomb
```

Generates a random catacomb.
